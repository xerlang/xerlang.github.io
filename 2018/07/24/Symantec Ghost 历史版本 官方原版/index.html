Symantec Ghost 历史版本 官方原版 【更新至12.0.0.11331】

发表于 2018-07-24 更新于 2021-02-20 分类于 数码

前言

Ghost一直是用着最趁手备份工具，由于Ghost本身官方并不提供下载，只集成在其他软件中，且官方下载相对困难。故将绝大部分版本的Ghost提取出来，以供下载。

此过程的难点在于搜集Symantec Ghost Solution Suite、Symantec Ghost Standard Tools、Symantec Deployment Solution与Ghost版本的对应关系及历史版本的官方原版程序的下载，提取起来很简单。

本文首发于卡饭论坛（https://bbs.kafan.cn/thread-2128366-1-1.html）及本博客，转载请注明出处。

说明

  • 本文所提供的Symantec Ghost Solution Suite、Symantec Ghost Standard Tools、Symantec Deployment Solution均为官方原版，均下载自Symantec官方服务器，除个别外，均带Symantec数字签名

  • 本文所提供的Ghost均为官方原版，均提取自官方Symantec Ghost Solution Suite、Symantec Ghost Standard Tools、Symantec Deployment Solution，除CLI程序（gdisk.exe、ghost.exe）外，GUI程序均带Symantec数字签名

  • 本文所提供的Ghost包含gdisk（gdisk.exe、gdisk32.exe、gdisk64.exe）与ghost（ghost.exe、ghost32.exe、ghost64.exe、ghostexp.exe、ghostexp64.exe、ghostsrv.exe等）相关程序

  • 本文所提供的Ghost包含8.x、11.x及12.x版本（12.0.0.7x版本前因找不到官方GSS、GST或GDS，未提供下载）

  • 因官方不再提供CLI程序，故本文所提供Ghost历史版本中的CLI程序（gdisk.exe、ghost.exe）均为未注册版，功能受限，GUI程序功能完整

新版说明

Symantec Ghost 12.0.0.11331提取自Symantec Deployment Solution 8.6。

更新日志

  • 自11.0.5.2113版本开始，官方开始提供gdisk、ghost的64位程序（gdisk64.exe、ghost64.exe）
  • 自12版本开始，官方不再提供CLI程序（gdisk.exe、ghost.exe）
  • 自12版本开始，官方开始提供ghostexp的64位程序（ghostexp64.exe）
  • 自12.0.0.8050版本开始，官方开始提供ghostexp、ghostsrv简体中文文件（ghostexp_zh.dll、ghostsrv_zh.dll）
  • 自12.0.0.8065版本开始，Ghost开始分卷压缩
  • 自12.0.0.10618版本开始，官方开始提供ghostsrv的64位程序（ghostsrv64.exe）
